SECTION 5: DATA INTEGRATION PROCESS
================================================================================

Files Used:
1. orders.csv - Transactional data (10,000 orders)
2. users.json - User master data (3,000 users)
3. restaurants.sql - Restaurant master data (500 restaurants)

Merge Process:
Step 1: Loaded orders.csv with 10,000 rows
Step 2: Loaded users.json with 3,000 user records
Step 3: Parsed restaurants.sql and extracted 500 restaurant records
Step 4: Performed LEFT JOIN on orders → users (on user_id)
        Result: Maintained all 10,000 orders, matched with user details
Step 5: Performed LEFT JOIN on result → restaurants (on restaurant_id)
        Result: Final dataset with all order, user, and restaurant information

Output: final_food_delivery_dataset.csv (10,000 rows × 16 columns)
